# 🍴 Food_trip

## 📌 プロジェクト概要
本プロジェクトは、沖縄県内の魅力的なグルメ情報を紹介することを目的としたWebサイトです。
ユーザーが直感的に「美味しそう！」と感じるビジュアル重視の設計に加え、店舗の詳細情報までスムーズにアクセスできる構成になっています。

* **開発期間**: 2025年10月 〜 2026年2月（予定）
* **開発人数**: 2名（チーム開発）
* **担当範囲**: フロントエンド開発・バックエンド開発
* **開発形態**: 進級制作

## 🎯 主な機能

### 🏠 メインページ（一覧表示）
* **カード型レイアウト**: アイキャッチ画像と短い紹介文を組み合わせたカード形式で、記事を視覚的に一覧表示。
* **プレビュー機能**: 記事をクリックする前に、投稿日やカテゴリ、魅力が伝わるキャッチコピーを確認可能。
* **プロフィール表示**: 投稿者の情報（Foodist プロフィール）を表示し、サイトの信頼性と親近感を向上。

### 📄 紹介詳細ページ
* **マルチフォト表示**: 複数の写真を用いて、料理や店舗の雰囲気を多角的に紹介。
* **体験レポート**: 実際に店舗を訪れた際の感想（「もちもち最強」など）を詳しく掲載。
* **店舗データ**: 店名、住所、営業時間などの実用的なデータをセクション分けして集約。
